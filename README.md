# AmazinStuff
A simple Balatro mod that adds some of my ideas. This mod requries JenLib as well as CG223's RarityLib.

## Contents

### Jokers
1 Awful Joker  
3 Common Jokers  
8 Uncommon Jokers  
8 Rare Jokers  
2 Pristine Jokers  
4 Legendary Jokers  
1 Mythic Joker  

### Consumeables
2 Spectral Cards
